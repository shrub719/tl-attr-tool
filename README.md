# Tomodachi Life (3DS) Song Tool

A command line tool for editing the Tomodachi Life 3DS Concert Hall songs.

## Usage

Still very WIP, but in case I forget:

```
main [file] [attrSet]
    file
        MSBT file extracted to TXT (e.g. using MSBT Editor).
    attrSet
        Which set of attribute definitions to use.
        0 - Song, 1 - Food
```
